## Featured Projects

### [TISER Multilingual RAG](https://github.com/arianmo477/tiser-multilingual-rag)

Multilingual temporal-reasoning system extending TISER across English, Italian, German and French.

- Qwen2.5-3B fine-tuning with QLoRA
- NLLB-200 multilingual translation pipeline
- FAISS-based retrieval-augmented generation
- Designed to run on an 8 GB GPU

`Python` `PyTorch` `Transformers` `QLoRA` `FAISS` `RAG`

---

### [Credit Card Fraud Detection](https://github.com/arianmo477/credit-card-fraud-detection)

End-to-end fraud-detection pipeline comparing supervised, unsupervised and deep-learning approaches on 284,807 transactions.

- Isolation Forest, One-Class SVM, LOF and KNN
- Autoencoder and Variational Autoencoder
- Cost-sensitive evaluation and threshold optimisation
- Random, chronological and nested cross-validation experiments

`Python` `scikit-learn` `PyTorch` `Pandas` `Anomaly Detection`

---

### [EgoCentricVisionPolito](https://github.com/arianmo477/EgoCentricVisionPolito)

End-to-end egocentric video-language system for video moment retrieval and generative question answering.

- Temporal moment localisation
- Video-language representation learning
- Multimodal generative question answering
- Experiments with VSLNet, EgoVLP, BERT and Video-LLaVA

`Computer Vision` `PyTorch` `LLMs` `Video Understanding`

---

### [Last Race](https://github.com/arianmo477/last_race)

Full-stack route-planning game developed with a React frontend, Express backend and SQLite database.

- User authentication and session management
- Route and adjacency validation
- Timed planning mechanics
- Persistent game state and scoring

`React` `JavaScript` `Node.js` `Express` `SQLite`
